The figure on the right shows a plot of \(P(d)\) vs. \(d\) for which \(P_0 = 1000\) photons and for which the probability that a photon is absorbed in each of \(50\) increments in distance is \(0.08\). The shape of this data suggests we can model the number of photons that remain at any distance, \(P(d)\), using an exponential decay:

\[\text{model 1: }P(d) = P_0 e^{-kd}\]

where \(P_0\) is the initial number of photons and \(k\) is a decay constant that is related to the probability that a photon is absorbed.

Exercise 1. Starting with the equation for model 1, derive the equations below for model \(2\) and for model \(3\), which are two other ways to express the relationship between \(P(d)\) and \(d\).

\[\text{model 2: } \ln(P(d)) = \ln(P_0) - kd\]

\[\text{model 3: } \ln \left( \frac{P(d)}{P_0} \right) = - kd\]
